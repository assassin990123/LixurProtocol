### What is Lixur ?
Welcome, we're glad to have you! Lixur is an open-sourced project building a distributed ledger platform. Lixur plans on implementing things such as DAGs and A.I smart contracts which enables fast, feeless, infinitely scalable, quantum resistant and an interconnected ecosystem with no-code SDKs, which users can make use of for any application requiring decentralized, automated intelligence and more.
